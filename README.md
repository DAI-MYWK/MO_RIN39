# RecruitMO UI プロトタイプ.

リクルートモバイルオーダーシステムのUIパターン比較ツールです。飲食店のメニュー情報をWebから収集し、複数のUIパターンを比較して最適なデザインを選択できます。

## 機能

1. **メニュー情報の収集**
   - 食べログ、ホットペッパーグルメからメニュー情報を自動収集
   - 手動でのメニュー追加・編集も可能

2. **UIパターンの表示**
   - パターン1: カテゴリ重視型 - カテゴリごとに整理されたメニュー表示
   - パターン2: 写真重視型 - 大きな写真でメニューを魅力的に表示
   - パターン3: おすすめ重視型 - おすすめメニューを前面に表示
   - パターン4: グリッドレイアウト - 均等なグリッドで整理された表示

3. **パターン比較機能**
   - サイドバイサイド表示で複数パターンを同時比較
   - 単一表示で詳細確認
   - 各パターンを拡大表示可能

## 技術スタック

- **Next.js 14** (App Router)
- **TypeScript**
- **Tailwind CSS**
- **Cheerio** (Webスクレイピング)
- **Framer Motion** (アニメーション)
- **Lucide React** (アイコン)

## セットアップ

```bash
# 依存関係のインストール
npm install

# 開発サーバーの起動
npm run dev
```

ブラウザで [http://localhost:3000](http://localhost:3000) を開いてください。

## 使い方

1. **メニュー収集タブ**
   - 食べログまたはホットペッパーグルメのURLを入力
   - 「収集」ボタンをクリックしてメニュー情報を取得
   - 必要に応じてメニュー情報を編集・追加
   - 「保存」ボタンでデータを保存

2. **UIパターン表示タブ**
   - 4つのUIパターンから選択
   - モバイル画面サイズでプレビュー表示

3. **パターン比較タブ**
   - サイドバイサイド表示で複数パターンを比較
   - 各パターンをクリックして拡大表示
